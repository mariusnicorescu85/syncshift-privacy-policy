‚è∞ Using the App
How do employees clock in/out?
On Shopify POS:
  • Tap the "POS Time Tracking Extension" tile on the home screen
  • Select employee name from dropdown
  • Choose action: Clock In, Clock Out, Break, or Lunch
  • Tap submit - done!
From Admin Dashboard:
  • Go to Apps ‚Üí POS Time Tracking
  • Click "Add Time Entry"
  • Fill in employee, action, and time details
Can employees track breaks and lunch?
Yes! The app tracks four types of actions:
  • Clock In: Start of work shift
  • Clock Out: End of work shift
  • Break: Short breaks during shift
  • Lunch: Lunch break tracking
Break and lunch times are automatically deducted from total work hours.
What if someone forgets to clock out?
Easy fixes:
  • Managers can add missing entries from the admin dashboard
  • Edit existing entries to correct times
  • Delete duplicate or incorrect entries
  • View all entries with timestamps for verification

How are hours calculated?
Hours are calculated automatically:
  • Work time = Clock Out time - Clock In time
  • Break time is tracked separately
  • Lunch time is tracked separately
  • Net work hours exclude break/lunch time
  • Overtime is flagged for shifts over 8 hours

üõ† Troubleshooting
The POS tile isn't showing up
Quick fixes:
  • Ensure app is installed in Shopify admin
  • Restart the POS app completely
  • Check POS is connected to internet
  • Verify POS app is updated to latest version
  • Contact support if tile still missing
Time entries aren't appearing in dashboard
Check these common issues:
  • Refresh the dashboard page
  • Verify date range filter includes entry dates
  • Check staff filter isn't excluding entries
  • Ensure POS has internet connection for sync
  • Wait 1-2 minutes for data synchronization
Hours calculation seems wrong
Common calculation issues:
  • Missing clock out entries (shows as 0 hours)
  • Clock out before clock in (invalid sequence)
  • Multiple clock ins without clock out
  • Time zone differences in manual entries
Use the edit feature to correct any timing issues.
